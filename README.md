# 🚀 Postly – AI Social Media Publishing Bot

Postly is a Telegram-based content publishing system that uses AI to generate and publish posts across multiple platforms (Twitter, LinkedIn, Instagram, Threads).

---

## ⚙️ Local Setup

```bash
git clone <repo-url>
cd postly-backend

# install deps
npm install

# start services
docker-compose up -d  # Redis

# run server
npm run dev
